📊 Amazon Sales Excel Project

📌 Project Overview

This project analyzes an Amazon Sales dataset using Microsoft Excel. The goal is to clean the data, create meaningful visualizations, and build an interactive dashboard that provides insights into sales performance, products, customers, and business trends.

🎯 Objectives

- Clean and prepare the sales data
- Analyze sales performance
- Identify top-performing products and categories
- Track profit and revenue trends
- Create an interactive dashboard for better decision-making

🛠️ Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Formulas and Functions
- Dashboard Design

📈 Dashboard Features

- Total Sales
- Total Profit
- Total Orders
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Top Selling Products
- Interactive Filters (Slicers)

📂 Dataset

The dataset contains Amazon sales records, including:

- Order ID
- Order Date
- Product
- Category
- Sales
- Profit
- Quantity
- Region
- Customer Details

💡 Key Insights

- Identified the highest-selling product categories.
- Analyzed monthly sales trends.
- Compared regional sales performance.
- Highlighted the most profitable products.
- Built an interactive dashboard for easy business analysis.
